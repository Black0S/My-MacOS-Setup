# Starship Theme: Fluent Minimal ∞

This is a clean, cinematic Starship prompt theme, elegant aesthetic and full Nerd Fonts support.

---

## Installation

1. Install **Nerd Fonts** for proper icons ([JetBrainsMono Nerd Font recommended](https://www.nerdfonts.com/#home)).
2. Add this configuration to your Starship config file (`~/.config/starship.toml`).
3. Restart your terminal and enjoy the **Fluent Minimal ∞** clean prompt.
